# 是否要加分号

## 自动插入分号规则
- 要有换行符，且下一个符号是不符合语法的，那么就尝试插入分号。
- 有换行符，且语法中规定此处不能有换行符，那么就自动插入分号。
- 源代码结束处，不能形成完整的脚本或者模块结构，那么就自动插入分号。

## PS： 不写分号需要注意的场景
- 以括号开头的语句。```(function(){})();(function)```
- 以数组开头的语句.```var a = []; [1,23,4]...```
- 以正则表达式开头的语句.``` var a = 1; /a/g...  ```
