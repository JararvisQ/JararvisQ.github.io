# 选择排序 O(n^2)
每次选出最小/大的元素放在第一位。

# 插入排序 O(n^2)
确保前部分的元素不变，每次循环拿出后面的元素往前扫瞄，反复将已排序的元素向后娜位，为tar提供插入空间。

# 冒泡排序 O(n^2)
两层嵌套，每次查看相邻元素，如果逆序，则交换。

# 快排 O(nLogN)
取标杆pivot，将小元素放左边，大元素放右边，然后依次对左，右数组继续快排，达到整个数组有序。

# 归并排序 O(nLogN)
把长度为n的数组，取一半，然后分别采用归并排序，最后将排序好的子序列合并成一个最终的排序序列。

# 堆排序 O(nLogN)
取最大值，最小值。依据数组依次建立小顶堆，然后依次取堆顶元素。
