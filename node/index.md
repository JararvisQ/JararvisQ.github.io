# Node.js
![](/image/b189097f73f1ab79690a5c9a5fa24d8.png)
- Node.js 是一个基于 chrome v8 引擎的JS运行环境
- Node.js使用一个事件驱动，非阻塞IO模型。

# Node 应用可以用来做什么
- web服务
  - 搜索引擎优化 + 首屏速度优化 -> 服务端渲染
  - 服务端渲染，前后端同构
- 工作流
- 可扩展性 - 庞大的社区技术支持
- 客户端应用
  - 客户端技术（electron），最大限度地复用工程

# BFF层 - backend for frontend
- 对用户侧提供 HTTP 服务
- 使用后端的 RPC 服务
