# 自顶向下解析HTTP1.1

## HTTP定义
- 无状态，应用层的，请求 / 应答，语义可扩展的，自描述消息，超文本系统。

## OSI模型与TCP/IP模型
![](/image/6bcedb1b71c458ebc3b79b53ba42008.png)
- 优点：方便更改当前层级的结构。
- 缺点：一个传输需要经过多个层级，降低效率。

## HTTP解决什么问题?
- world wild web。主要解决人与机器通讯的问题。
- 为了解决www信息交互面对的需求：
  - 低门槛。
  - 可扩展性，巨大的用户群体，超长的寿命。
  - 分布式系统下的超文本媒体数据，大粒度的网络传输。（视频，音乐）
  - 不可预测流量负载，非法格式的数据，恶意信息。
  - 不可保持多个请求间的状态信息。
  - 新老组件并存，向前兼容。
